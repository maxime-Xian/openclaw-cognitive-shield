# 项目管理说明文档 (max-cognitive-shield)

## 1. 项目规划 (深度重构版)
本项目的核心目标是构建一个**元认知操作系统 (Meta-Cognitive OS)**，将 AI 从被动的助手升级为主动的**前额叶监护人 (Prefrontal Guardian)**。

### 核心机制：
1.  **主动脉冲 (Heartbeat)**：通过定时心跳实现睡眠守护、晨间简报、晚间总结及自动错误闭环。
2.  **双通道意图过滤**：精准区分用户真实意图与技术背景，确保干预的准确性。
3.  **三级分级干预 (L1/L2/L3)**：基于风险与频率的阶梯式干预协议。
4.  **进化型记忆架构**：通过 `memory/` 目录下的反射、进化与纠错模块，实现 AI 的自我迭代。
5.  **模块化思维模型**：集成决策矩阵、风险评估及根因分析等专业框架。

### 核心任务列表：
1.  **架构深度解析**：重新审视 `core/` 目录下的 SOUL, AGENTS, HEARTBEAT 等核心协议。
2.  **文档升维**：将 README 从“技能介绍”提升为“操作系统级架构说明”。
3.  **推广方案优化**：重点突出“元认知”与“前额叶监护”这一独特技术护城河。
