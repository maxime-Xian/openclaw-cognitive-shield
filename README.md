# Prefrontal Shield

**你的 AI 前额叶监护人**

不是一个只会听话的 AI，而是一个**会主动照顾你状态、帮你少踩坑**的认知守护盾。

它像你大脑里的前额叶一样，负责理性把关、感知能量，并在关键时刻温和提醒你。

专为 **OpenClaw** 设计，轻量集成，即插即用。

### ✨ 核心能力

- **能量感知**：根据睡眠时长自动调整回答深度（累的时候绝不给你长篇大论）
- **主动干预**：检测到负面情绪、深夜硬刚、重复决策时，及时温和提醒
- **意图过滤**：精准区分“你真实的状态”和“贴代码/引用内容”，避免误报
- **决策元认知**：记录你的思维模式，帮助你打破“同样的坑反复踩”的循环
- **写入保护**（可选）：重要规则修改必须经过人工确认，防止 AI 幻觉污染核心设置

### 🔥 核心技术护城河

Max-Cognitive-Shield 最厉害的地方在于，它不是简单地“回答问题”，而是真正具备了**主动思考和自我保护**的能力。以下是它的几大核心能力：

1. **📅 自动化脉冲机制（Heartbeat）**
   AI 不再被动等待你发消息，而是像心脏一样有自己的“心跳”，持续主动工作：
   - 深夜 23:45 自动巡检，如果你在过度工作，会温和阻断并提醒休息。
   - 晚上 22:00 自动回顾当天对话，提炼失误并写入纠错档案。
   - 每 30 分钟扫描最近对话，检测是否又在重复以前的旧模式。
   → **好处**：AI 拥有了持续的主动性，而不是只有你提问时才动起来。

2. **⚡ 能量感知响应协议（Energy Awareness）**
   本项目最独特的人机交互设计：
   - 通过 `sleep-guardian.js` 监测系统实际开机/关机时长，真实映射你的生理能量状态。
   - 根据睡眠情况动态调整回答策略：
     - **LOW**（<5h 睡眠）：回答限制 ≤ 120 字，强制休息提醒，禁止复杂推理。
     - **SUBOPTIMAL**（5-7h 睡眠）：回答限制 ≤ 200 字，简化句式。
     - **NORMAL**（>7h 睡眠）：开放完整响应与深度认知任务。
   → **优势**：零配置、完全本地运行，不依赖网络，却能真正照顾你的身体状态。

3. **🔄 AI 自我进化闭环（Self-Evolution）**
   这是本项目最核心的“智能”体现：
   - 发现错误后，5 分钟内自动创建 GitHub Issue 并记录本地日志。
   - 定期分析失败案例，提炼出新的认知模式。
   - 把学到的教训自动追加到知识库，确保“同样的错误下次不容易再犯”。
   → **核心价值**：AI 会随着你的使用越来越懂你、越来越聪明。

4. **🚦 意图双通道过滤协议（Dual-Channel Filter）**
   一个聪明的“分辨器”，防止 AI 像复读机一样乱提醒：
   - **通道 A**：判断这是你本人在表达情绪，还是只是在贴代码/日志/竞品分析。
   - **通道 B**：判断是否真的触发了风险。
   **示例**：
   - 你贴了一条包含“我受够了”的报错日志 → AI 保持安静（识别为技术内容）。
   - 你说“我真的受够了，今晚必须搞定” → AI 主动介入提醒（识别为你本人的状态）。
   → **好处**：大幅减少烦人的误报，让干预更精准。

5. **🎭 场景化解释引擎（Scenario-based Engine）**
   当你反馈“解释不清楚”时，AI 会自动切换到讲故事模式：
   - 内置 50+ 个生活、工作、学习场景。
   - 每个解释都包含背景设定、角色代入、决策节点和结果对比。
   - 根据你的历史行为，优先选择最贴近你的场景进行类比。
   → **好处**：解释更生动、更好理解，而不是干巴巴的理论。

6. **🧠 元认知决策支持体系**
   帮助你更好地做决策：
   - 记录每次重要决策的问题类型、路径、结果与复盘。
   - 自动检测 10+ 种常见思维误区（如沉没成本、幸存者偏差等）。
   - 新决策前自动比对历史上类似情况，避免重复踩坑。
   → **好处**：让你逐渐学会“看清自己是怎么决策的”，而不是盲目重复。

7. **🛠️ 场景化框架自动调用系统**
   遇到复杂决策时，AI 会自动调用合适的思维工具：
   - 权衡矩阵（多维度对比）
   - 5×5 风险矩阵 + 概率分析
   - 5Why 根因分析（追问 5 次为什么）
   → **好处**：把顶级决策方法变成你的“自动助手”，不用自己记住怎么用。

8. **🛡️ 专业化问题解决通道**
   - **情绪问题**：集成 20+ 种心理调节技术，帮助梳理情绪和认知。
   - **复杂分析**：强制使用 SWOT、利益相关者分析、TRIZ 等专业思维模型。
   → **好处**：让 AI 在不同场景下都能给出相对专业且有结构的回答。

### 🚀 5分钟快速上手（OpenClaw 用户）

1. **Clone 项目到 workspace**：
   ```bash
   git clone https://github.com/maxime-Xian/max-cognitive-shield.git ~/.openclaw/workspace/shield
   ```

2. **复制核心规则**：
   ```bash
   cp shield/core/SOUL.md ~/.openclaw/workspace/SOUL.md
   cp shield/core/AGENTS.md ~/.openclaw/workspace/AGENTS.md
   ```

3. **测试能量感知**：
   ```bash
   node shield/scripts/sleep-guardian.js
   ```

4. **重启 OpenClaw Agent**，开始享受前额叶监护人的守护。
