# 马克西姆揍钢琴 (MaximeXian) 品牌 VI 手册

## 1. 品牌核心理念
- **名称**: 马克西姆揍钢琴
- **定位**: 元认知操作系统与高效能生活方式引领者
- **口号**: 像仲裁者一样思考，像监护人一样行动

## 2. 视觉规范
### 2.1 色彩系统
- **主色**: 深邃蓝 (`#1A2B4C`) - 代表智慧、稳重、元认知
- **辅助色**: 能量金 (`#FFD700`) - 代表警示、能量、进化
- **背景色**: 纸张灰 (`#F5F5F5`) - 代表冷静、简洁、非煽情

### 2.2 字体规范
- **中文字体**: 苹方 (PingFang SC) / 思源黑体 (Source Han Sans)
- **英文字体**: Inter / Helvetica Neue
- **字号原则**: 正文 ≥ 14px，标题 18-24px

### 2.3 图标与 Emoji
- **核心 Emoji**: 🧠 (元认知), 🚦 (干预), 📅 (脉冲), ⚡ (能量)
- **风格**: 线性、极简、高对比度

## 3. 对外话术标准
- **人设**: 冷静的仲裁者，深邃的监护人
- **原则**: 高价值密度、结构化、禁止无谓煽情、禁止过度承诺
