# 隐私影响评估报告 (Privacy Impact Assessment, PIA)

## 1. 评估背景
本报告旨在评估 **Max-Cognitive-Shield** 对用户隐私数据处理的合规性，确保符合 GDPR 与《个人信息保护法》。

## 2. 数据处理活动清单
| 数据类型 | 用途 | 加密方式 | 存储期限 |
|---------|------|---------|---------|
| 用户 ID (微信/平台) | 身份识别 | SHA-256 | 长期 |
| 会议标题/链接 | 主动提醒 | AES-256 | 任务完成后立即删除 |
| 情绪评分/运动步数 | 量化日报 | 聚合处理 | 30天 |
| 错误日志 | 自动进化 | 脱敏处理 | 7天 |

## 3. 安全控制措施
- **加密**: 所有敏感字段在数据库中均采用 AES-256 加密。
- **最小化**: 仅采集实现提醒与进化所必须的最小数据项。
- **匿名化**: 错误闭环日志在上传至 GitHub Issue 前进行脱敏。

## 4. 评估结论
当前系统架构在数据加密、存储时限及采集范围方面表现良好，符合国际与国内主流隐私保护法规。
